# SimplyToDo

<h3> View the Website <a href="https://simplytodo1.netlify.app/">Live</a></h3>
<h2> Project Description</h2>
<p>
Introducing SimplyToDo - the ultimate solution for keeping track of your tasks. With this user-friendly app, you can effortlessly manage both completed and pending tasks with ease. Say goodbye to the hassle of keeping track of your to-dos and hello to a simplified, organized life!
</p>
